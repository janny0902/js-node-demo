# js-node-demo

## 프로젝트 개요
- Node.js와 lowdb를 활용한 간단한 데이터베이스 예제 프로젝트

## 주요 파일
- index.js: 메인 서버 코드
- db.json: lowdb를 사용하는 JSON 기반 데이터베이스 파일
- Dockerfile: 도커 환경 지원
- test/: 테스트 코드 디렉토리

## 실행 방법
1. Node.js 설치
2. 의존성 설치: `npm install`
3. 서버 실행: `node index.js`

## 특징
- lowdb를 활용한 파일 기반 DB 예제
- Dockerfile 포함으로 컨테이너 환경 지원
- 테스트 코드 포함

- 학습 및 데모 목적의 간단한 Node.js 프로젝트
